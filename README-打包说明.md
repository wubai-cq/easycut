# 轻松切 - 打包说明

## 📦 打包脚本说明

### 🎯 主要脚本

| 脚本名称 | 功能 | 适用场景 |
|---------|------|----------|
| `打包菜单.bat` | 主菜单，选择打包方式 | 推荐使用 |
| `build.bat` | 完整打包脚本（详细版） | 正式发布 |
| `build-quick.bat` | 快速打包（仅安装包） | 快速测试 |
| `build-portable-only.bat` | 便携版打包 | 绿色使用 |
| `build-all.bat` | 完整打包（安装包+便携版） | 正式发布 |
| `build-clean.bat` | 清理构建文件 | 释放空间 |

### 🚀 使用方法

#### 方法1：使用菜单（推荐）
```bash
# 双击运行
打包菜单.bat
```

#### 方法2：直接运行脚本
```bash
# 完整打包
build.bat

# 快速打包
build-quick.bat

# 便携版
build-portable-only.bat
```

#### 方法3：命令行
```bash
# 安装包
npm run build:win

# 便携版
npm run build:portable

# 全部
npm run build:all
```

### 📋 生成的文件

#### 安装包
- **文件**：`dist/轻松切-1.0.0.exe`
- **特点**：
  - 自动安装到系统
  - 创建桌面快捷方式
  - 支持卸载
  - 进程名称：轻松切

#### 便携版
- **文件**：`dist/win-unpacked/轻松切.exe`
- **特点**：
  - 无需安装，直接运行
  - 可复制到其他电脑
  - 进程名称：轻松切
  - 绿色软件

### 🔧 环境要求

- **Node.js**：v16+ 
- **npm**：v8+
- **操作系统**：Windows 10/11
- **权限**：管理员权限（推荐）

### ⚠️ 注意事项

1. **首次打包**：需要安装依赖，时间较长
2. **网络要求**：需要下载 Electron 二进制文件
3. **磁盘空间**：至少需要 1GB 可用空间
4. **杀毒软件**：可能误报，需要添加信任

### 🐛 常见问题

#### 问题1：依赖安装失败
```bash
# 使用国内镜像
npm config set registry https://registry.npmmirror.com
```

#### 问题2：打包失败
```bash
# 清理后重试
build-clean.bat
build.bat
```

#### 问题3：进程名称显示为 Electron
- 开发模式：正常现象
- 打包后：会正确显示为"轻松切"

### 📊 文件大小参考

- **安装包**：约 150-200MB
- **便携版**：约 200-250MB
- **源码**：约 50MB

### 🎯 最佳实践

1. **开发阶段**：使用 `build-quick.bat`
2. **测试阶段**：使用 `build-portable-only.bat`
3. **正式发布**：使用 `build-all.bat`
4. **定期清理**：使用 `build-clean.bat`

### 📞 技术支持

如果遇到问题，请检查：
1. Node.js 版本是否正确
2. 网络连接是否正常
3. 磁盘空间是否充足
4. 杀毒软件是否拦截
