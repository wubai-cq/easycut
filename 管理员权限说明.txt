======================================
  管理员权限设置说明
======================================

【重要】本应用需要管理员权限才能正常工作！

一、安装后自动设置（推荐）
======================================
安装程序已自动配置快捷方式以管理员身份运行。
安装完成后，通过以下方式启动应用即可：

1. 双击桌面快捷方式
2. 从开始菜单启动

这些快捷方式已经默认以管理员身份运行，会自动弹出 UAC 提示。

二、手动设置（如果自动设置失败）
======================================
如果启动后无法禁用/启用网卡，请按以下步骤手动设置：

方法 1：设置快捷方式属性
1. 右键点击桌面上的"网络切换工具"快捷方式
2. 选择"属性"
3. 点击"高级"按钮
4. 勾选"用管理员身份运行"
5. 点击"确定"保存

方法 2：直接以管理员身份运行
1. 右键点击快捷方式
2. 选择"以管理员身份运行"

方法 3：设置可执行文件属性
1. 打开安装目录（通常在 C:\Program Files\网络切换工具\）
2. 找到"网络切换工具.exe"文件
3. 右键点击 → 属性
4. 切换到"兼容性"标签
5. 勾选"以管理员身份运行此程序"
6. 点击"确定"

三、验证管理员权限
======================================
启动应用后，如果看到以下情况之一，说明已获得管理员权限：

✓ 窗口标题栏显示"管理员"字样
✓ 能够成功禁用/启用网卡
✓ 控制台日志正常显示命令执行结果

如果出现以下情况，说明没有管理员权限：

✗ 禁用/启用网卡时报错
✗ 提示"权限不足"
✗ 网卡状态无法改变

四、常见问题
======================================
Q: 为什么需要管理员权限？
A: 禁用/启用网卡需要执行系统命令（netsh），
   这些命令必须在管理员权限下才能运行。

Q: 每次启动都要确认 UAC 提示吗？
A: 是的，这是 Windows 安全机制的要求。
   快捷方式已设置为管理员运行，但仍需手动确认。

Q: 可以关闭 UAC 提示吗？
A: 不建议关闭 UAC，这会降低系统安全性。
   如果确实需要，可以在 Windows 设置中调整 UAC 级别。

Q: 安装后快捷方式图标有盾牌标志吗？
A: 如果快捷方式正确设置，应该在图标右下角显示盾牌标志。
   如果没有，请使用上述"手动设置"方法。

五、技术细节
======================================
- 安装程序类型：NSIS
- 安装位置：C:\Program Files\网络切换工具\
- 快捷方式位置：桌面 + 开始菜单
- 权限要求：requireAdministrator
- 执行命令：netsh interface set interface

如有问题，请查看应用的控制台日志输出。
