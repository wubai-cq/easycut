# EasyCut 激活系统测试指南

## 测试流程

### 第一次启动（需要激活码）

1. **重置激活状态**（模拟首次安装）：
   ```bash
   node test-reset.js
   ```
   应该看到：`✅ 重置成功！现在启动应用应该会显示激活界面`

2. **启动应用**：
   ```bash
   npx electron .
   ```
   应该看到：激活界面，包含激活码输入框和"激活许可证"按钮

3. **输入激活码**：
   - 使用测试激活码：`N1Y6-7W44-JY3Q-E28A`
   - 点击"激活许可证"按钮
   - 应该看到：成功消息，然后自动跳转到主应用界面

4. **验证主应用界面**：
   - 应该看到：主应用界面（快速切换按钮、网络接口区域）
   - 网络接口区域应该自动加载，显示网络接口列表
   - 状态栏应该显示"就绪"

5. **测试刷新按钮**：
   - 点击"刷新"按钮
   - 应该看到：网络接口重新加载，但**不会**回到激活界面

### 第二次启动（不需要激活码）

1. **直接启动应用**：
   ```bash
   npx electron .
   ```
   应该看到：直接进入主应用界面，不需要输入激活码

2. **验证功能**：
   - 网络接口应该自动加载
   - 刷新按钮应该正常工作
   - 所有功能应该正常使用

## 修复的问题

### ✅ 问题1：激活后网络接口不自动加载
**原因**：激活成功后没有正确调用 `loadInterfaces()` 函数
**修复**：在 `activateApp()` 函数中添加了延迟调用 `loadInterfaces()`

### ✅ 问题2：刷新按钮回到激活界面
**原因**：刷新按钮调用了 `window.location.reload(true)`，导致页面重新加载
**修复**：改为只调用 `loadInterfaces()` 函数，不重新加载页面

### ✅ 问题3：主应用界面显示问题
**原因**：激活成功后没有正确调用 `showMainApp()` 函数
**修复**：在激活成功后明确调用 `showMainApp()` 函数
