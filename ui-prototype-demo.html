📖 设计说明

🎯

深色科技风

采用深海军黑背景 (#0E1122) 配合青绿强调色 (#39e6cf),营造专业的网络管理工具氛围

✨

流畅交互

使用 cubic-bezier 缓动曲线,悬停提升、点击缩放、光晕扩散等微交互,让每次操作都有反馈

🎨

语义化颜色

绿色表示成功/启用,红色表示危险/禁用,蓝色表示信息/连接,清晰直观

📱

中文优化

使用 Microsoft YaHei 字体,字号、行高、字重都针对中文界面优化

⚡

状态可视化

通过颜色徽章、动画条、图标旋转等视觉元素,实时反映网络状态
