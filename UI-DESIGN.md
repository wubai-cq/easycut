## 4. 交互流程设计

### 4.1 激活流程

```
启动应用
  ↓
显示激活界面
  ↓
用户输入激活码
  ↓ (格式化: XXXX-XXXX-XXXX-XXXX)
点击"激活许可证"
  ↓
验证中... (按钮禁用 + 加载动画)
  ↓
┌─────────┬─────────┐
│ 成功    │ 失败    │
├─────────┼─────────┤
│ ✓ 激活成功!│ ✗ 激活码无效│
│ 自动跳转  │ 可重试   │
└─────────┴─────────┘
  ↓
进入主界面
```

### 4.2 网络切换流程

```
点击"启用外网"按钮
  ↓
按钮提升 + 阴影加深 (悬停反馈)
  ↓
点击瞬间: 缩放至 98% (点击反馈)
  ↓
按钮禁用 + 显示加载状态
  ↓
执行网络切换命令
  ↓
┌──────────┬──────────┐
│ 成功     │ 失败     │
├──────────┼──────────┤
│ 更新状态  │ 显示错误  │
│ 绿色指示器│ 恢复按钮  │
└──────────┴──────────┘
  ↓
更新网络接口列表
  ↓
状态栏显示结果
```

### 4.3 网络接口管理流程

```
加载网络接口列表
  ↓
显示"加载中..."占位符
  ↓
获取接口信息
  ↓
渲染接口卡片列表
  ↓
┌──────────────┐
│ 每个接口卡片: │
├──────────────┤
│ 显示名称      │
│ 显示状态      │
│ 显示状态徽章  │
│ 提供操作按钮  │
└──────────────┘
  ↓
用户点击"启用/禁用"
  ↓
按钮反馈 (缩放 + 波纹)
  ↓
执行操作
  ↓
更新徽章 + 状态文本
```

### 4.4 刷新流程

```
点击"更新"按钮
  ↓
图标旋转 360° (0.4s 动画)
  ↓
重新加载接口列表
  ↓
状态栏显示"更新中..."
  ↓
更新完成
  ↓
状态栏显示"已更新"
```

### 4.5 分享流程

```
点击分享图标
  ↓
图标放大 1.1x + 青绿发光
  ↓
显示分享覆盖层
  ↓
┌───────────────────┐
│   [二维码图片]     │
│                   │
│ 公众号500佰       │
│ 欢迎留言 发送     │
│ EasyCut获取下载   │
│ 链接              │
└───────────────────┘
  ↓
点击任意位置关闭
```

## 6. 动画与过渡

### 6.1 标准动画

**按钮悬停:**
- 时长: 0.3s
- 缓动: cubic-bezier(0.4, 0, 0.2, 1)
- 效果: translateY(-3px) + scale(1.01)

**按钮点击:**
- 时长: 0.1s
- 缓动: ease
- 效果: scale(0.98)

**刷新图标旋转:**
- 时长: 0.4s
- 缓动: ease-in-out
- 效果: rotate(360deg)

**绿色指示条动画:**
- 时长: 2s
- 缓动: ease-in-out
- 循环: infinite
- 效果: background-position 移动

### 6.2 过渡效果

**颜色过渡:**
- 时长: 0.25s
- 属性: color, background-color

**阴影过渡:**
- 时长: 0.3s
- 属性: box-shadow

**变换过渡:**
- 时长: 0.3s
- 属性: transform

## 7. 状态反馈设计

### 7.1 按钮状态

**默认状态:**
- 正常显示
- 可交互

**悬停状态:**
- 提升效果
- 阴影加深
- 光晕效果

**激活状态:**
- 缩小效果
- 阴影减弱
- 波纹扩散

**禁用状态:**
- 灰度滤镜
- 降低透明度
- 禁止交互

**加载状态:**
- 禁用交互
- 显示加载动画 (可选)

### 7.2 网络状态

**已连接:**
- 蓝色徽章
- 禁用按钮可用

**已禁用:**
- 红色徽章
- 启用按钮可用

**未连接:**
- 灰色徽章
- 启用按钮可用

### 7.3 系统状态

**就绪:**
- 状态栏: "就绪"
- 所有功能可用

**处理中:**
- 状态栏: "处理中..."
- 相关按钮禁用

**成功:**
- 状态栏: "操作成功"
- 自动恢复就绪

**失败:**
- 状态栏: "操作失败"
- 显示错误信息

## 8. 无障碍设计

### 8.1 键盘导航

- Tab 键: 在可交互元素间切换
- Enter 键: 激活当前元素
- Escape 键: 关闭弹窗/覆盖层

### 8.2 焦点指示

- 蓝色焦点环 (输入框)
- 提升效果 (按钮)
- 光晕效果 (图标按钮)

### 8.3 颜色对比

- 主要文本: 18:1 对比度 (AAA)
- 次要文本: 15:1 对比度 (AAA)
- 徽章文本: 4.5:1 对比度 (AA)

### 8.4 语义化标签

- 使用正确的 HTML5 标签
- ARIA 标签增强可访问性
- 图标配备 alt 文本

---

## 9. 性能优化

### 9.1 渲染优化

- 使用 CSS transforms (GPU 加速)
- 避免 layout thrashing
- 虚拟滚动 (长列表)

### 9.2 动画优化

- 使用 will-change 提示浏览器
- 限制动画元素数量
- 使用 requestAnimationFrame

### 9.3 资源优化

- SVG 图标 (小文件)
- 压缩图片资源
- 延迟加载非关键资源
