## 3. 交互行为

### 3.1 打开设置面板

1. 点击顶部操作栏的设置图标 ⚙️
2. 图标旋转动画 (360°, 0.4s)
3. 遮罩层淡入 (0.3s)
4. 模态框从中心缩放弹出 (0.3s, ease-out)
5. 加载保存的设置项

### 3.2 关闭设置面板

**方式 1: 点击关闭按钮 ✕**
- 模态框缩放淡出 (0.2s)
- 遮罩层淡出 (0.3s)

**方式 2: 点击遮罩层**
- 模态框缩放淡出 (0.2s)
- 遮罩层淡出 (0.3s)

**方式 3: 按 ESC 键**
- 模态框缩放淡出 (0.2s)
- 遮罩层淡出 (0.3s)

### 3.3 修改设置项

**开关切换:**
1. 点击开关
2. 开关滑块动画移动 (0.3s)
3. 背景颜色变化 (0.3s)
4. 不立即保存,等待用户点击"保存"按钮

**单选切换:**
1. 点击选项
2. 选项圆点变化 (0.2s)
3. 标签颜色变化 (0.2s)

**下拉选择:**
1. 点击下拉框
2. 展开下拉菜单 (0.2s)
3. 选择选项
4. 收起下拉菜单 (0.2s)

### 3.4 保存设置

1. 点击"保存"按钮
2. 按钮显示加载状态 (禁用 + 旋转图标)
3. 保存设置到 localStorage / 配置文件
4. 显示成功提示 (Toast)
   - "✓ 设置已保存"
   - 2秒后自动消失
5. 关闭设置面板
6. 应用新设置

### 3.5 重置设置

1. 点击"重置"按钮
2. 显示确认对话框
   - 标题: "确认重置"
   - 内容: "确定要重置所有设置为默认值吗?"
   - 按钮: [取消] [确认]
3. 点击确认
   - 恢复默认值
   - 显示提示 "✓ 已恢复默认设置"
4. 点击取消
   - 关闭对话框

### 3.6 清除缓存

1. 点击"清除缓存"按钮
2. 显示确认对话框
   - 标题: "确认清除"
   - 内容: "确定要清除所有缓存数据吗?"
   - 按钮: [取消] [确认]
3. 点击确认
   - 清除缓存
   - 显示提示 "✓ 缓存已清除"

## 4. 动画效果

### 4.1 模态框动画

**打开:**
```css
@keyframes modalIn {
    from {
        opacity: 0;
        transform: scale(0.9) translateY(-20px);
    }
    to {
        opacity: 1;
        transform: scale(1) translateY(0);
    }
}
```

**关闭:**
```css
@keyframes modalOut {
    from {
        opacity: 1;
        transform: scale(1) translateY(0);
    }
    to {
        opacity: 0;
        transform: scale(0.9) translateY(-20px);
    }
}
```

### 4.2 开关动画

```css
@keyframes toggleSlide {
    from {
        transform: translateX(0);
    }
    to {
        transform: translateX(20px);
    }
}
```

### 4.3 设置图标旋转

```css
@keyframes settingsSpin {
    from {
        transform: rotate(0deg);
    }
    to {
        transform: rotate(360deg);
    }
}
```

## 7. 设置存储

### 7.1 存储方式

**localStorage:**
```javascript
localStorage.setItem('easycut-settings', JSON.stringify(settings));
```

**配置文件 (Electron):**
```javascript
const { app } = require('electron');
const path = require('path');
const fs = require('fs');

const configPath = path.join(app.getPath('userData'), 'settings.json');
fs.writeFileSync(configPath, JSON.stringify(settings, null, 2));
```

### 7.2 加载设置

```javascript
function loadSettings() {
    try {
        const saved = localStorage.getItem('easycut-settings');
        if (saved) {
            return JSON.parse(saved);
        }
    } catch (error) {
        console.error('加载设置失败:', error);
    }
    return getDefaultSettings();
}
```
