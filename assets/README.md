# Assets 目录

此目录用于存放应用的资源文件。

## 图标文件

在打包应用之前，请准备以下图标文件：

### icon.png
- 用途：应用窗口图标
- 推荐尺寸：256x256 或 512x512 像素
- 格式：PNG

### icon.ico
- 用途：Windows 可执行文件图标
- 推荐尺寸：包含多个分辨率（16x16, 32x32, 48x48, 256x256）
- 格式：ICO

## 创建图标的方法

### 方法一：在线工具
1. 使用 [icoconvert.com](https://icoconvert.com/) 将PNG转换为ICO
2. 上传你的PNG图标（建议512x512）
3. 选择包含多个分辨率
4. 下载生成的ICO文件

### 方法二：使用工具
- Windows: 使用 IcoFX 或 GIMP
- 在线: Favicon Generator, CloudConvert

### 方法三：使用默认图标
如果没有自定义图标，可以暂时移除 `package.json` 中的 icon 配置：

```json
"build": {
  "win": {
    // 移除或注释掉这一行
    // "icon": "assets/icon.ico"
  }
}
```

## 图标设计建议

- 使用简洁的设计，避免过于复杂的细节
- 考虑在小尺寸下的显示效果
- 使用高对比度的颜色
- 网络相关主题：可以使用网络、连接、WiFi等元素

## 示例图标主题

网络切换工具的图标可以包含以下元素：
- 🌐 地球/网络符号
- 🔄 切换箭头
- 📡 信号波纹
- 🔌 连接插头
- 💻 电脑设备
