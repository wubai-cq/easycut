## 使用示例

### 有效激活码示例
```
N1Y6-7W44-JY3Q-E28A
C5T3-6P91-PT7O-VY77
T3R2-0G77-LS6T-WNLQ
E5T7-8B16-CK0X-62IY
I5V2-5Z99-EQ2M-WG8W
```

### 激活流程
1. 启动应用
2. 如果未激活，显示激活界面
3. 输入16位激活码（自动格式化）
4. 点击"激活许可证"或按回车
5. 激活成功后自动进入主应用

## 安全特性

1. **机器绑定**：基于MAC地址的唯一机器标识
2. **格式验证**：严格的激活码格式校验规则
3. **防重复使用**：同一激活码在同一机器只能使用一次
4. **本地存储**：激活状态仅存储在本地，无需网络连接
5. **输入验证**：前端和后端双重验证

## 存储位置

激活状态存储在：`C:\Users\[用户名]\.easycut\activation.json`

文件内容示例：
```json
{
  "activated": true,
  "machineId": "b4:45:06:7b:16:33",
  "activationCode": "N1Y67W44JY3QE28A",
  "activatedAt": "2025-10-22T05:08:33.764Z"
}
```
